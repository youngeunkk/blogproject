# MINI BLOG

## 🎫프로젝트 소개
<br>
나만의 공간이라는 느낌이 드는 다이어리 형식의 블로그입니다!  

새로고침 없이 부드럽게 동작하기 위해 React.js를 사용했습니다.


<br>

## 🎉주요 기능
<br>

* Create, Read, Update, Delete 기능이 구현되었습니다.
* Home 화면에서 날짜 보기가 가능합니다.

<br>

## 🎈프로젝트 사용 방법
